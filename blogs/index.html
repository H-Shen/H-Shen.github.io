Welcome to my Blog Section!

This blog is a collection of some of my coding experience and skills used in coding competitions as well as solutions for some coding problems that I found interesting.

One thing you will notice is that almost all the code for solving problems in competitive programming is written in C++, why? Well, here are some reasons:

  • C++ is faster and easier to tweak than other languages in real coding competitions.

  • C++ has STL, which implements a lot of built-in data structures and algorithms and thus brings much convenience to coders.

  • You can directly use CPU instructions or inline assembly code in C++ for optimization, even though such case is pretty rare.

There are other advantages of using C++ in coding competitions since it can help you get in touch with the hardware more easily than other languages.

Anyway, unlike those WIKI algorithms, this blog just contains some techniques or introductions of some data structures I found useful and not that hard to understand for a CS student. Happy reading!
